  2. BrowserMap

    June 2012 - present

    BrowserMap is a JavaScript browser features detection library. It uses modular probes - code snippets that detect specific features of the client; these are then used to detect the client's type. Once the client has been identified, BrowserMap can forward it to alternate website versions, which are more suitable for the type of device used to access the application. Since 2013 the project has been donated to the Apache Software Foundation, as part of Apache DeviceMap.
