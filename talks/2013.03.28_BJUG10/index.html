1. If it didn't happen on the dev@list it didn't happen.

2. Whatever you're working on, it must be backed by an issue in the tracker.

3. If it's not in the SCM system, it doesn't exist.

4. If it's important, it needs a permanent URL.

5. What happened when you weren't here? Check the archives or the live feeds!
